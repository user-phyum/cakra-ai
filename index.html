Konfigurasi API Key

Panduan Pembuatan Gemini API Key
1

Buka Google AI Studio

Kunjungi Google AI Studio lalu masuk dengan akun Google Anda.

2

Masuk ke Menu API Key

Klik tombol Get API Key di panel kiri, kemudian klik tombol hijau Create API Key.

3

Pilih Project Google Cloud

Jika muncul pilihan, klik Create API key in a new project (atau pilih project yang sudah ada), lalu klik tombol konfirmasi.

4

Salin Kunci Anda

Salin kode acak berawalan AIzaSy... yang muncul di layar Anda.

5

Tempel ke Kolom Input

Tempel (*paste*) kode API Key yang telah dicopas tersebut ke dalam kolom input "Google Gemini API Key" di bagian bawah panduan ini, lalu klik "Simpan & Hubungkan".

CATATAN: Kunci ini bersifat rahasia. Jangan membagikannya ke publik karena siapa pun yang memiliki key tersebut dapat menggunakan kuota API Anda.

Aplikasi ini berkomunikasi langsung dengan model Cakra-Ai dan Cakra-Ai TTS. Masukkan API Key Google AI Studio Anda di bawah ini. Kunci Anda akan disimpan dengan aman di localStorage peramban Anda.

Tentang Cakra-Ai Lite

Cakra-Ai Lite adalah asisten mandiri yang cerdas. Berbeda dengan chatbot biasa, Cakra-Ai di sini hanya akan memfokuskan pemikirannya pada materi/dokumen sumber yang Anda unggah.

Daftar Fitur Utama:
🎯 Konteks Terisolasi: Bertanya spesifik dari data unggahan Anda.
🎙️ Input Suara (STT): Menggunakan dikte suara Bahasa Indonesia bawaan premium.
🔊 Read Aloud Premium: Menggunakan suara orisinal buatan Cakra-Ai TTS.
📥 Unduh & Ekspor: Simpan respons dan diskusi lengkap ke berkas Microsoft Word (.doc) terstruktur.
⏱️ Timestamp Presisi: Semua pesan serta rekaman suara (transkrip rapat) dilacak dengan penanda waktu riil.

Versi 1.3.5 • Dikembangkan khusus dengan Cakra-Ai.
